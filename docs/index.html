Quick Installation

1

Add the APT repository

echo "deb [trusted=yes] https://zales.github.io/sys-ink/ ./" | sudo tee /etc/apt/sources.list.d/sys-ink.list
2

Update and install

sudo apt update && sudo apt install sys-ink
3

Done!

The service starts automatically after installation. View logs with:

journalctl -u sys-ink -f
